The purpose of this project was to take digital photos using adequate resolution, effective lighting, and interesting composition.

Original Photo (left): Taken looking up into the inside of a clock tower, used composition technique of vantage point, indoor lighting, captured on iPhone 13 using the automatic settings, the resolution is 72 ppi.

Edited Photo (right): Editing work done in Photoshop, I set the resolution to 300 ppi, cropped and rotated it to better frame the image, used the auto setting to adjust the levels, used clone stamp tool to remove the glare from the lights.

Original Photo (left): Taken in outdoor afternoon lighting while it was cloudy outside, used composition concept of proximity, captured on iPhone 13 using the automatic settings, the resolution is 72 ppi.

Edited Photo (right): Editing work done in Photoshop, set image resolution to 300 ppi, auto-adjusted the levels, used auto color and auto tone.

Original Photo (left): Taken in indoor studio lighting, used asymmetrical composition and rule of thirds, captured on iPhone 13 using automatic settings, the resolution is 72 ppi.

Edited Photo (right): Editing work done in Photoshop, set the image resolution to 300 ppi, cropped the top of the image to eliminate some of the unwanted background space, used the automatic levels adjustment, used the polygonal lasso tool to select the background, then darkened this background area by lowering the brightness and increased the contrast of the background.

Original Photo (left): Taken looking up through the center of a stairwell, used concept of vantage point and leading lines, indoor lighting, captured on iPhone 13 using automatic settings, the resolution is 72 ppi.

Edited Photo (right): Editing done in Photoshop, set resolution to 300 ppi, cropped the bottom of the image to center it better, used the auto levels adjustment.

Original Photo (left): Taken in cloudy outdoor afternoon lighting, used composition technique of having a close proximity to the subject, captured on iPhone 13 using automatic settings, the resolution is 72 ppi.

Edited Photo (right): Editing done in Photoshop, set resolution to 300 ppi, used the auto levels adjustment, used auto color and auto contrast.

Original Photo (left): Taken in outdoor afternoon lighting while it was cloudy, used asymmetrical composition and rule of thirds, captured on iPhone 13 using automatic settings, the resolution is 72 ppi.

Edited Photo (right): Editing done in Photoshop, set resolution to 300 ppi, auto-adjusted the levels, cropped the image to better use the rule of thirds for the placement of the swans, used auto color and auto tone, and adjusted the color balance, shifting it to be more green.
